## 安装 tailwindcss

```
$ pnpm install -D tailwindcss@latest postcss@latest autoprefixer@latest

$ pnpm tailwindcss init -p
Created Tailwind CSS config file: tailwind.config.js
Created PostCSS config file: postcss.config.js
```
